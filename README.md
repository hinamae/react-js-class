##　トラブルシューティング

asdfを使用しているために出るエラー

```
asdf: No version set for command npx
you might want to add one of the following in your .tool-versions file:

nodejs 12.18.3

```
### 対処
```
$HOME/.tool-versions
に
nodejs 12.18.3
の行を付け足した。

```
